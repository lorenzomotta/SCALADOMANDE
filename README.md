# Scala Domande

Gioco quiz in stile "Chi vuol essere Milionario" con 10 domande per argomento.

## Avvio locale

```bash
npm install
npm run dev
```

## Supabase

Esegui `supabase_setup.sql` nel SQL Editor di Supabase.  
Puoi usare lo **stesso progetto** di Quattro Parole.

## Password admin

Default: `VALHALLA` (modificabile in `App.tsx`)
